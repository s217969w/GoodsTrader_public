# グッズ交換支援アプリ

## 開発目的

缶バッジやトレーディングカードは所持数が増えるにつれ頭の中での管理が大変になると思います。特に交換する場面では、自分が何を持っていて何が欲しいのか、お互いに確認する作業から入ることが多いです。
そこで、おすすめの交換方法を提案してもらい、円滑なトレードを実現するためのweb上で動くアプリを開発しています。

## 重要: 使用する前に
グッズのデータ、名前などをまとめ、`src/data/cardinfo.ts`に記述してから使用してください。また、`src/components/Menu/SideMenu.tsx`、`src/pages/portal.tsx`にてサイト名を設定しています。各自変更してからご使用ください。

## 実行方法
Node.jsの使用方法と同様です。
環境が構築されていることを確認し、下記のコマンドを実行してください。
```
npm install
```
開発サーバーの起動は下記のコマンドでできます。
```
npm run dev
```
